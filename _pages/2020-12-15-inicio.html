ActiveScaffold

Constructor rápido de CRUD
para Rails

Ahorra tiempo y dolores de cabeza, y crea un conjunto de páginas más fácil de mantener, con ActiveScaffold.

ActiveScaffold gestiona todas tus necesidades de interfaz de usuario CRUD (crear, leer, actualizar, eliminar), dejándote más tiempo para centrarte en problemas más desafiantes (¡e interesantes!).

¿Qué es ActiveScaffold?

ActiveScaffold es una gema de Ruby on Rails que simplifica la creación de interfaces de usuario completas para operaciones CRUD (Crear, Leer, Actualizar, Eliminar).

Incluye funciones adicionales como búsqueda, paginación y controles de diseño, lo que permite a los desarrolladores centrarse en aspectos más complejos de sus aplicaciones.

Empieza rápidamente

1. Añade la gema a tu proyecto:

					 gem 'active_scaffold'

2. Instala las dependencias:

					bundle install

3. Genera un recurso con ActiveScaffold:

					rails generate active_scaffold:resource ModelName

4. Migra la base de datos:

					rails db:migrate

5. Arranca el servidor y accede a la interfaz:

					rails server

Visita http://localhost:3000/model_names para ver ActiveScaffold en acción.
