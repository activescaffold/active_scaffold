Bridges
=======

Extend and integrate ActiveScaffold functionality effortlessly.

Bridges
-------

Bridges are connectors that integrate ActiveScaffold with other Rails gems. They enable seamless communication between ActiveScaffold and external libraries, making it easier to add features like user authentication, file uploads, or PDF generation.

Unlike plugins, bridges depend on both ActiveScaffold and the external gem they connect with, working together to provide extended functionality.

### Paperclip Bridge​

Provides seamless support for file attachments using Paperclip, enabling users to upload and manage files directly through ActiveScaffold.

### CarrierWave Bridge​

Integrates CarrierWave for managing file uploads, allowing developers to handle file storage and display effortlessly.

### Dragonfly Bridge

Supports Dragonfly for dynamic asset processing, including image resizing, encoding, and other on-the-fly operations.

Why use Bridges?
----------------

### Expand Features

Add powerful new functionalities like batch processing, advanced search, and file uploads.

### Integrate Seamlessly

Connect with popular Rails gems like Devise, Paperclip, and CarrierWave.

### Save Time

Avoid reinventing the wheel by leveraging pre-built extensions.

### Customize Easily

Tailor plugins and bridges to fit the unique requirements of your application.

How to Get Started
------------------

1. Add the plugin or bridge gem to your Gemfile. For example:
```
gem 'paperclip'
```
2. Run bundle install to install the dependencies:
```
bundle install
```
          

3. Configure the plugin or bridge in your application according to the documentation.

4. Restart your Rails server and enjoy the new features.
